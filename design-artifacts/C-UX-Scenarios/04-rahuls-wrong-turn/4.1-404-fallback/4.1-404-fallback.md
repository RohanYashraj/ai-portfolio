## Overview

**Page Purpose:** Hold the gallery's voice and credibility off the map. Identity intact, a smart suggestion mapping the dead URL to its likely new home, one-tap paths to the main rooms. The best 404 is never seen: known legacy paths 301-redirect before this renders.

**User Situation:** Rahul opened a months-old bookmark to the previous site pre-hiring-panel; trust at its most fragile — a generic 404 confirms the staleness fear.

**Success Criteria:** Visitor reaches live content in one tap; zero generic-error vocabulary; legacy redirect map catches known old-site paths so this page renders only for genuinely unknown URLs.

**Entry Points:**
- Stale bookmarks, dead external links/search results to the previous site, mistyped URLs

**Exit Points:**
- Smart suggestion → likely Archive entry (primary) · room links → any room · DocentLauncher

## Technical Notes

- **Redirect map first:** enumerate all URLs of rohanyashraj.github.io at cutover → 301 to new homes; this page is only for unmapped paths. Returns HTTP 404 + noindex
- Fuzzy match: path tokens vs. CMS slugs, threshold-gated (a wrong confident guess is worse than none)
- Log 404 hits — recurring unknown paths feed the redirect map (living-gallery maintenance loop)
