## Overview

**Page Purpose:** Make a time-poor evaluator form the credibility hypothesis in under one second and pull them into the work. Three proofs before scrolling: **who** (Dr. Rohan Yashraj Gupta), **what** ("the AI Actuary" — actuary FIA/FIAI × data scientist), **how much** (key stats). Exit job: momentum into Selected Works.

**User Situation:** A credible pointer (peer recommendation, LinkedIn post, conference bio) put this URL in front of them seconds ago. Evaluative, impatient, mildly skeptical. Hope: "show me quickly this person is real." Worry: "another buzzword portfolio."

**Success Criteria:** Content visible <1s on mid-range mobile; visitor can state the positioning after one viewport; scroll-through rate into Selected Works.

**Entry Points:**
- Google search ("Rohan Gupta actuary AI", "AI Actuary")
- LinkedIn profile link (mobile-heavy)
- Conference materials (bios, slide footers, QR codes)
- Direct URL / bookmark; internal: header logo from any room; 404 recovery

**Exit Points:**
- Scroll (primary) or room-cue tap → Selected Works
- Room nav → any room · Docent suggested tap → DocentPanel

## Page States

| State | When | Appearance | Actions |
|-------|------|------------|---------|
| Default | Normal load | Full hero, stats counted up | All |
| First-paint | 0–1s | Name/niche/anchor render instantly (system-font fallback OK); stats show final values if JS pending | Scroll |
| Reduced-motion | prefers-reduced-motion | Static stat numbers, no type-in effects | All |
| Docent-off | Agent disabled | Launcher hidden entirely; page fully functional | All |
| Stats-unavailable | CMS fetch fails | Stat strip hidden (never zeros/placeholders) | All |

## Technical Notes

- <1s content paint on mid-range Android: hero is static HTML/SSG, zero blocking scripts; count-up is progressive enhancement
- "AI Actuary" SEO ownership: H1/meta/schema.org Person + JSON-LD; this page carries the Engine goal
- Stats sourced from CMS singleton (auto-derivable counts preferred over manually maintained numbers)
- Docent launcher lazy-loads after content paint
