## Overview

**Page Purpose:** Make the actuary→AI path imaginable. One warm human paragraph (exams → pricing team → AI), the site's single great portrait, and inline links to the first paper and first talk. Narrative warmth after the proof rooms.

**User Situation:** Ananya, 9pm after study, arrived from a shared conference talk. Evaluating the *person*: is this path walkable by someone with my credentials? Elena also skims here for bio material.

**Success Criteria:** The story reads in under a minute; the path lands as concrete steps, not mystique; ≥1 inline artifact link or docent tap follows.

**Entry Points:**
- Room nav ("About") · docent hand-off · LinkedIn bio links · Elena grabbing speaker-bio material

**Exit Points:**
- Docent suggested question → DocentPanel (primary for Ananya)
- Inline links → first paper / first talk case details · room nav

## Technical Notes

- One portrait, used once — enforced by convention: no other page object references a headshot
- Story is a single CMS rich-text field; path markers a CMS list — both feed docent knowledge
- Tone guard: the story must name concrete artifacts (first paper, first model), not qualities
